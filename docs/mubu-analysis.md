# 幕布快捷键与编辑功能分析

## 一、幕布核心功能梳理

### 文本样式快捷键
| 幕布快捷键 | 功能 | Markdown 对应 |
|-----------|------|--------------|
| Ctrl+B | 加粗 | `**文字**` |
| Ctrl+I | 斜体 | `*文字*` |
| Ctrl+U | 下划线 | 无原生支持 |
| Alt+颜色字母 | 文字变色 | 无原生支持 |

### 结构操作快捷键
| 幕布快捷键 | 功能 | 说明 |
|-----------|------|------|
| Enter | 新建同级项 | 幕布核心操作 |
| Tab | 缩进（降级） | 创建子项 |
| Shift+Tab | 提升层级 | 上移层级 |
| Ctrl+Shift+U | 提升层级 | 同 Shift+Tab |
| Ctrl+Shift+D | 降级 | 同 Tab |
| Ctrl+↑ | 上移当前项 | 调整顺序 |
| Ctrl+↓ | 下移当前项 | 调整顺序 |
| Ctrl+Enter | 展开/折叠 | 大纲导航 |

### 操作快捷键
| 幕布快捷键 | 功能 |
|-----------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+C/V | 复制/粘贴 |
| Ctrl+X | 剪切 |
| Ctrl+A | 全选 |
| Ctrl+S | 保存 |
| Ctrl+F | 搜索 |
| Ctrl+N | 新建文档 |
| / | 快捷命令面板 |

### 右键菜单功能
- 插入图片
- 插入链接
- 添加标签
- 任务清单
- 代码块
- 分割线
- 高亮背景色

## 二、我们工具的核心定位差异

| 维度 | 幕布 | 我们的工具 |
|------|------|-----------|
| 定位 | 大纲笔记工具 | Markdown 文件浏览器 |
| 核心 | 结构化编辑 + 思维导图 | 阅读为主 + 轻量编辑 |
| 文件格式 | 专有格式 | .md 标准 Markdown |
| 主要用户 | 笔记、知识管理 | 文件浏览、快速查看 |

## 三、建议整合的功能（按优先级）

### ✅ 强烈建议整合

| 功能 | 原因 | 实现方式 |
|------|------|---------|
| Ctrl+B 加粗 | 用户最常用，降低学习成本 | 自动包裹 `**` |
| Ctrl+I 斜体 | 同上 | 自动包裹 `*` |
| Tab/Shift+Tab 缩进 | 列表编辑核心操作 | 调整 Markdown 缩进 |
| Enter 换行 | 基础操作 | 智能换行 |
| Ctrl+Z/Y 撤销重做 | 编辑必备 | 浏览器原生 |
| / 快捷命令 | 幕布特色，降低学习成本 | 弹出常用语法列表 |
| 任务清单 | 小白最常用的 Markdown 功能之一 | 自动生成 `- [ ]` |
| 高亮背景色 | Markdown 颜色，实用 | 用 `<mark>` 标签 |

### 🔄 建议整合（需适配）

| 功能 | 原因 | 注意事项 |
|------|------|---------|
| Ctrl+Shift+U/D 层级调整 | 结构编辑利器 | 需适配 Markdown 列表语法 |
| 颜色文字 | 视觉标记 | Markdown 不原生支持，需 HTML |
| 图片插入 | 实用功能 | 需文件选择器 |
| 链接插入 | 基础功能 | 自动包裹 `[]()` |
| 高亮背景色 | 视觉标记 | 用 `<mark>` |

### ❌ 不建议整合

| 功能 | 原因 |
|------|------|
| 大纲折叠/展开 | 我们是文件浏览器，不是大纲工具 |
| 思维导图 | 偏离核心定位 |
| 实时协作 | 需要后端服务，太重 |
| 云同步 | 我们是本地工具 |
| 代码块语法高亮编辑 | 编辑器内做代码高亮太复杂 |
| Ctrl+Enter 新建层级 | 幕布特有，Markdown 没有对应概念 |

## 四、Office 模式设计方案

### 模式定位
Office 模式 = Word 般的所见即所得 + Markdown 语法自动转换

### 核心交互

```
用户操作             →  Markdown 转换
─────────────────────────────────────
Ctrl+B 选中文字     →  **文字**
Ctrl+I 选中文字     →  *文字*
Tab 缩进           →  增加 2 空格 / 调整列表层级
/ 打开命令面板      →  显示常用语法快捷列表
鼠标选中+工具栏     →  对应 Markdown 语法
```

### 工具栏设计

```
[加粗] [斜体] [删除线] | [标题▼] [列表] [任务] | [链接] [图片] [代码块] [高亮] | [撤销] [重做]
```

### 建议采用的快捷键

